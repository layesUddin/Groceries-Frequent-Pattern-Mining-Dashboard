# Groceries Frequent Pattern Mining Dashboard

Interactive dashboard for mining frequent itemsets and association rules from grocery transaction data. Built with Streamlit — adjust support/confidence thresholds and explore results in real time.

## Dataset

Source: [Groceries Dataset on Kaggle](https://www.kaggle.com/datasets/irfanasrullah/groceries)

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
